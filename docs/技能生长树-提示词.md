# 个人技能生长树 — AI 生成提示词



## 项目背景（一句话）

这是一个「职业能力大数据服务平台」的前端项目（Vue 3 + Vite），需要为学生的个人主页做一个**技能生长树**可视化组件。

---

## 需求描述

请帮我生成一个**独立的 HTML 文件**（包含 CSS + JS，可直接在浏览器打开），实现一棵**逼真的个人技能生长树**。

### 核心概念

- 这是一棵**会生长的大树**，代表学生的技能成长历程
- 学生每学会一项新技能，树上就**长出一条新枝丫**
- 枝丫末端显示**技能名称**
- 支持**手动添加技能**（输入技能名 + 选择类别）

### 视觉要求（非常重要）

1. **树要逼真**
   - 粗壮的树干，有树皮纹理（深棕色渐变 + 纵向纹理线）
   - 树根分叉扎入地面
   - 树冠顶部有小分枝
   - 地面有草地/泥土效果

2. **枝丫要自然**
   - 从树干不同高度自然分叉出来
   - 使用贝塞尔曲线（quadraticCurveTo），不要直线
   - 左右交替生长，角度随机但有规律
   - 枝丫从粗到细（树干端粗，末端细）
   - 每条枝丫上有 1-2 个小分枝

3. **粒子效果（重点）**
   - 枝丫生长时，生长点有**发光粒子拖尾**
   - 枝丫长完后，末端**爆发粒子**（像开花/结果）
   - 背景有**飘浮的微光粒子**（萤火虫效果）
   - 粒子颜色与技能类别颜色一致

4. **技能标签**
   - 枝丫末端显示技能名称（白色文字 + 发光光晕）
   - 下方显示类别小标签（彩色）
   - 末端有一个发光的小圆点（像果实/花苞）

5. **配色方案**
   - 背景：深蓝黑色调（`#0a0e27` 到 `#141e30`），模拟夜空
   - 树干：暖棕色系（`#3d2b1f`、`#6b4c3b`、`#8b6914`）
   - 技能类别颜色：
     - 编程语言：红色系 `#ff6b6b`
     - 框架：金色系 `#ffd93d`
     - 数据库：绿色系 `#6bcb77`
     - 运维工具：蓝色系 `#4d96ff`
     - 软技能：紫色系 `#c084fc`

6. **动画**
   - 枝丫生长动画：从树干出发，沿贝塞尔曲线延伸到终点（约 1-1.5 秒）
   - 生长完成后标签淡入
   - 完成后粒子爆发
   - 整个树有微弱的呼吸/摇摆动画（可选）

### 交互功能

- 底部有一个输入面板（半透明毛玻璃效果）：
  - 文本输入框：输入技能名称
  - 下拉选择框：选择技能类别（编程语言/框架/数据库/运维工具/软技能）
  - 「+ 添加技能」按钮
  - 「重置」按钮
- 支持回车键添加
- 顶部显示「已掌握 N 项技能」计数
- 最多支持 12-15 个技能节点

### 技术约束

- **纯 HTML + CSS + JS**，不依赖任何框架或库
- 使用 **Canvas** 绘制（不要用 SVG/DOM）
- 使用 `requestAnimationFrame` 做动画循环
- 响应式：`canvas` 尺寸随窗口变化
- 代码结构清晰，有注释
