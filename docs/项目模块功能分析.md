# 项目模块功能分析

> **创建日期**: 2026-07-15  
> **用途**: 全面梳理项目每个模块、每个文件的功能  
> **项目**: 职业能力大数据服务平台

---

## 一、项目概述

基于 SpringBoot + Vue 3 的多角色 SaaS 系统，服务四类角色：**学生、教师、管理员、企业 HR**。

---

## 二、后端模块（8 个 Maven 模块）

### 2.1 `occupation-common` — 公共基础模块

| 子包/文件 | 功能 |
|-----------|------|
| `ai/AiChatClient.java` | **全平台唯一 LLM 出口**，OpenAI 兼容协议，支持单轮/多轮/JSON模式 |
| `ai/AiProperties.java` | AI 配置（DeepSeek API、温度、Token 上限） |
| `ai/AiMessage.java` | 对话消息体（role + content） |
| `config/MyBatisPlusConfig.java` | 多租户插件 + 分页配置 |
| `config/KafkaTopicConfig.java` | Kafka Topic 定义 |
| `exception/BizException.java` | 业务异常类 |
| `exception/GlobalExceptionHandler.java` | 全局异常处理器（`@RestControllerAdvice`） |
| `dto/Result.java` | 统一响应 `{code, message, data}` |
| `resources/sql/init.sql` | **19 张表 + 种子数据**（系统初始化脚本） |

### 2.6 `occupation-recommend` — 推荐与 AI 核心模块（最大模块，118 个文件）

#### 2.6.1 Controller 层

| 文件 | 功能 | 角色 |
|------|------|------|
| `RecommendController.java` | 职位推荐列表、收藏、投递、联系雇主 | Student |
| `StudentResumeController.java` | 简历 CRUD + **AI 诊断/润色** | Student |
| `StudentAdvisorController.java` | **AI 职业顾问** 多轮对话 | Student |
| `StudentReportController.java` | **AI 学生报告**（6 大板块） | Student |
| `StudentProfileController.java` | 学生画像管理 | Student |
| `HrController.java` | 职位管理、人才浏览、投递处理 | HR |
| `TeacherController.java` | 学生管理、教学建议 | Teacher |

#### 2.6.2 Service 层 — AI 服务

| 文件 | 功能 |
|------|------|
| `ResumeAiService.java` | **AI 简历诊断**（review）+ **AI 简历润色**（polish）+ 多轮润色聊天 |
| `CareerAdvisorService.java` | **AI 职业顾问**（多轮对话 + 匹配理由解读 `explainMatch()`） |
| `StudentAiReportService.java` | **AI 学生报告**（6 大板块：技能评估、岗位匹配、薪资预测、学习建议、竞争力分析、职业规划） |
| `TeachingAiService.java` | **AI 教学建议**（5 大板块：班级分析、技能缺口、课程建议、学生预警、就业指导） |

#### 2.6.3 Service 层 — 推荐引擎

| 文件 | 功能 |
|------|------|
| `HybridRecommendService.java` | 混合推荐引擎（融合多路召回结果） |
| `CollaborativeFilterService.java` | 协同过滤推荐（基于用户行为相似度） |
| `ContentBasedRecommendService.java` | 内容推荐（基于职位-技能匹配） |
| `SemanticMatchService.java` | JD-简历语义匹配（计算匹配度分数） |

#### 2.6.4 Service 层 — 业务服务

| 文件 | 功能 |
|------|------|

| `JobApplicationService.java` | 投递管理（状态机：SUBMITTED→VIEWED→INTERVIEW→OFFER/REJECTED） |
| `HrJobService.java` | HR 职位管理 |
| `TeacherStudentService.java` | 教师端学生管理 |

#### 2.6.5 Mapper 层

                       │ HTTP (JWT Auth)

│  occupation-recommend│ 推荐引擎 + AI 服务（核心模块）      │

├─────────────────────────────────────────────────────────┤
│                   occupation-common                      │

└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│   MySQL 8.0  │  Redis 6.2  │  Kafka 7.5  │  Nginx 1.24  │
