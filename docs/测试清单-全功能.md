## 八、端到端业务流程测试

### 8.1 管理员完整流程

```
1. 登录 → 查看数据看板（5个图表正常渲染）
2. 创建采集任务 → 启动任务 → Mock模拟采集 → 查看采集日志
3. 手动重算分析数据 → 看板数据更新
4. 创建报告模板 → 生成报告 → 下载报告（HTML格式）
5. 创建学生账号 → 编辑用户 → 禁用/启用用户
6. 查看报告列表 → 删除报告
```

### 8.2 学生完整流程

```
1. 登录 → 完善个人画像（专业/技能/城市/薪资）
2. 查看个性化推荐（验证匹配分数+匹配原因+缺失技能）
3. 点击职位 → 查看详情 → 收藏职位
4. 返回推荐页 → 投递另一个职位
5. 查看我的收藏（验证已收藏职位存在）
6. 查看求职统计（APPLY:1, FAVORITE:1）
7. 查看推送消息（投递后自动生成推送通知）
8. 标记消息已读 → 未读数归零
```

### 8.3 教师完整流程

```
1. 登录 → 查看班级概览（统计卡片+学生列表）
2. 点击学生 → 查看行为统计+行为明细
3. 搜索特定学生
4. 查看教学建议（技能缺口+改革建议+趋势图）
```

### 8.4 HR 完整流程

```
1. 登录 → 查看工作台（统计卡片+最近职位+人才推荐）
2. 发布新职位 → 查看职位列表
3. 编辑职位信息
4. 浏览人才库 → 搜索/筛选人才 → 查看详情
5. 下架职位
```
