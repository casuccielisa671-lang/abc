## 五、系统级功能（跨角色）

### 5.1 认证与多租户
| 功能 | 说明 |
|------|------|
| JWT 登录 | 用户名+密码+租户名 → JWT Token，BCrypt 密码加密 |
| 多租户隔离 | MyBatis-Plus 插件自动注入 `tenant_id`，行级数据隔离 |
| 角色权限 | ADMIN / TEACHER / STUDENT / HR 四角色，`@PreAuthorize` 注解鉴权 |
| 路由守卫 | Token + 角色双重校验，401 自动跳转登录页 |

### 5.2 对外开放 API
| 功能 | 说明 |
|------|------|
| Token 鉴权 | apiKey + apiSecret → accessToken（Redis 存储，30分钟过期） |
| 职位查询 | 开放职位检索接口 |
| 就业大盘 | 总岗位数/平均薪资/城市数/行业数（Redis 缓存10分钟） |
| 热门技能 | 词频统计 TopN（Redis 缓存10分钟） |
| 行业分布 | SQL GROUP BY 行业岗位分布 |
| 限流 | 基于 Token 的请求频率限制 |

### 5.3 推送通知
| 功能 | 说明 |
|------|------|
| 推送列表 | 分页查询，未读优先 |
| 标记已读 | 含归属校验 |
| 未读计数 | 红点提示 |

### 5.4 数据链路
```
数据采集（BOSS直聘/Mock）→ Kafka → 数据清洗（去重/标准化/校验）
→ job_detail（清洗后职位表）→ 分析重算（5维度聚合）
→ analysis_result（分析结果表）→ Dashboard图表 / 报告生成 / 职位推荐
```
