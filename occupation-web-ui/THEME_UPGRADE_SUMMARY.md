## 使用方法

### 切换主题
1. **登录页**: 右上角太阳/月亮按钮
2. **应用内**: 功能栏右边的主题切换按钮
3. 点击即可平滑切换到另一种模式

### 主题持久化
- 用户选择的主题保存在localStorage
- 下次访问自动应用上次选择的主题

### 浏览器支持
- 所有现代浏览器 (Chrome, Firefox, Safari, Edge)
- 支持CSS Variables (IE 11 通过Element Plus降级)
